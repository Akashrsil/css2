Google Font

Google Fonts (formerly known as Google Web Fonts) is a computer font and web font service owned by Google. This includes free and open source font families, an interactive web directory for browsing the library, and APIs for using the fonts via CSS and Android.Google Fonts Knowledge supplements our font catalog with a rich library of peer-reviewed guides to the world of typography, created by typographic experts around the world. It covers universal principles of typography for any medium, with scope to delve into all of the artistic and technical aspects of modern typesetting and font technology
